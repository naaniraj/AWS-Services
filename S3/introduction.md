Key Features:<br>
✅ Stores any type of file (videos, images, logs, backups, etc.).<br>

### What is the advantages of S3 ?
*************************************
1. **Scalability**: ---S3 is designed to scale with your needs, handling massive amounts of data.<br>
2. **No storage limits**: ---There's no maximum storage limit for an S3 bucket.<br>
3. **Object storage**:--- S3 stores data as objects, which can be up to 5 TB in size.<br>

1.**Cost**:--- Storing large amounts of data with significant costs.<br>
2.**Data retrieval**:--- Retrieving large amounts of data can take time and impact performance.<br>
3.**Data management**:--- Managing massive amounts of data requires careful planning and organization.<br>

1.**Use lifecycle policies**:--- Manage data retention and deletion to optimize storage costs.<br>
2.**Use versioning**:--- Track changes to objects and maintain data integrity.<br>
3**Use bucket policies**:--- Control access and security for your S3 bucket<br>
